# Shazaming_Whales
Building a convolution neural network (CNN) to classify 3 whale species given audio data collected by NOAA. Performed extensive data preprocessing and cleaning to collect and cut audio data into a usable format for the CNN.

This project is a collaborative project between Colby Anderson, Salem Burtner, and Scarlett Shropshire
